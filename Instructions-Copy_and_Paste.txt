This is a simple version of the instructions to simplify Copy and Paste to 5250.  
It is advisable to use this alongside the Instructions PDF.

N.B. If you download the new Change_HOME-Directory from 6th January 2026,
     having used the Original version, follow these instructions

If you do NOT change the USRPRF home directory
1. copy the new code into /tmp/ritfori/code 
2. CRTBNDCL PGM(QTEMP/AcmePgms)                                            
   SRCSTMF('/tmp/ritfori/code/AcmePgms.clle') DFTACTGRP(*NO) ACTGRP(*NEW)  
3. CALL QTEMP/ACMEPGMS

If you DO change the USRPRF home directory
Use a DIFFERENT user to delete the current DIR and library to start again
Now change the HOME directory of your user e.g. to /home/USRPRF
1. DLTLIB LIB(RITFORI)
2. RMDIR DIR('/USRPRF') SUBTREE(*ALL) 
3. RMDIR DIR('/tmp/USRPRF') SUBTREE(*ALL)
4. DEL OBJLNK('/tmp/RitFori-main.zip')
4. CHGUSRPRF USRPRF(USRPRF) HOMEDIR('/home/USRPRF') or whatever directory you want
5. Start from the beginning and download RITFORI etc
6. Once yor finished you can delete the OLD Certificate from the DCM

If starting for the first time, follow these instructions

Download instructions
https://github.com/RitFori/RitFori/blob/main/Instructions.pdf
  (just put the link into a browser and click on Download raw file button)

Recommended to use RITFORI as USRPRF, but can be anything 
   and HOMEDIR can be what you want, but do not create the last directory (usually /USRPRF)
Do not use special characters except for _ in USRPRF and PASSWORD

   CRTUSRPRF USRPRF(USRPRF) PASSWORD(password) CURLIB(RITFORI) SPCAUT(*ALLOBJ *SECADM *AUDIT) HOMEDIR('/home/usrprf') 

Download RITFORI code to your pc from https://github.com/RitFori/RitFori/archive/refs/heads/main.zip
 (just put the link into a browser or use Github - Code - Download Zip) 

Upload RitFori-main.zip to /tmp on the IBM i
 (ACS – Integrated File System is good for this)

5250 Signon as your USRPRF

    CD DIR('/tmp')
    CALL PGM(QP2TERM)
    PATH=/QopenSys/pkgs/bin:\$PATH
    export PATH PASE_PATH
    unzip ./RitFori-main.zip
    mv RitFori-main ritfori
   F3

5250 Still Signed on as your USRPRF

    CRTBNDCL PGM(QTEMP/AcmeSetup1) SRCSTMF('/tmp/ritfori/code/AcmeSetup1.clle') DBGVIEW(*SOURCE)
    CALL PGM(ACMESETUP1)

5250 Sign off and on again as above USRPRF

    ACMESETUP2 EMAIL('e-mail') CERTNAME('name.domain.TLD') TOKEN('"domain_token"') JKSPASS('jks_password')

Web (HTTP) Admin with user profile who has *IOSYSCFG
DCM Setup part 1 – Manual – see Instructions.pdf

    /HOMEDIR/acme/data/certs/name.domain.TLD_ecc/lekeystore.jks
    Jks_password
    name.domain.TLD

Browser

    https://name.domain.TLD:2012/openapi/ui/

DCM with user profile who has *ALLOBJ and *SECADM - can use your USRPRF 
DCM Setup part 2 – Manual – see Instructions.pdf

    your_application_name_for_DCM

5250 Still Signed on as your USRPRF

    DCM_SETUP3 CERTNAME('name.domain.TLD') USRPWD('usr_password') DCMPWD('DCM_password') CERTAPP('certificate_app_name') 

Renew the certificate before 90 days (Do not Renew it with DCM=YES on the same day)

    LERenew CERTNAME('name.domain.TLD') DCM_YN(YES) 

    CALL PGM(LE_ADMIN5) 
    DCMDltCert CERTNAME('name.domain.TLD')
